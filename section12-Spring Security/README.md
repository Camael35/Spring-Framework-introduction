## 十二、Spring Security
### 1. 什麼是Spring Security?
#### Spring Security是一個提供認證與授權的軟體架構
#### ○ 認證: 確保使用者人如其名。Spring Security提供認證的機制有Http Basic, Form Based … 等。
#### ○ 授權: 確保使用者僅可存取允許的資源。Spring Security提供幾種授權的層級，有http請求、http方法、物件等。
